## Usage Classification

- `dark/`: blue marks intended for white or light neutral surfaces.
- `light/`: white marks intended for dark SLU Blue or Grand Blue surfaces.
- `svg/`: canonical web SVG copies.
- `png/`: fallback raster assets.
- `optimized/`: web-ready SVG copies preserved without recoloring, cropping, or effects.

## Rules

- SVG is the default web format.
- PNG is a fallback only.
- Do not stretch, recolor, crop, shadow, outline, or add visual effects.
- Preserve aspect ratio and clear space.
- Use white variants only on dark institutional backgrounds.
